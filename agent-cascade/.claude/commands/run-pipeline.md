# Запуск пайплайна разработки

## Задача

Запустить цепочку "разработка → тестирование" по указанной спецификации.

## Алгоритм

1. Проверь, что файл `.agent_docs/specs/$ARGUMENTS.md` существует
2. Если файл не найден — сообщи пользователю и покажи список доступных спецификаций в `.agent_docs/specs/`
3. Если файл найден — передай задачу агенту `developer`:

```
Реализуй фичи по спецификации .agent_docs/specs/$ARGUMENTS.md
Следуй инструкциям из своего описания агента.
После завершения вызови агента tester для проверки.
Максимум 3 итерации фикс→тест.
После завершения всех итераций — сообщи мне финальный результат.
```

4. Дождись завершения и сообщи пользователю итог:
   - Сколько итераций потребовалось
   - Финальный вердикт (PASSED / FAILED с оставшимися проблемами)
   - Ссылки на созданные отчёты
